# genai-employee-engagement-assistant
A Generative AI-powered application that creates employee engagement activities and professional email announcements for workplace events using OpenAI LLMs and Gradio interface. Designed to automate HR content creation and improve workplace engagement planning.

## ▶️ How to Run (Locally)

This application is built using Gradio and OpenAI API.

To run locally:

1. Clone the repository
2. Install dependencies:
   pip install gradio openai
3. Set your OpenAI API key as an environment variable
4. Run:
   python app.py

Note: This project uses OpenAI API, so a valid API key is required.
